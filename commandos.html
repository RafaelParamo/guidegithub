Comandos de cambio en el tiempo

git checkout permite cambiarte a estados previos de commits si estas sobre la misma rama o bien cambiarte a otra rama git rm --cached archivo.txt saca de la lista de espera y los regresa a la fabrica a los archivos. git rm --force: Elimina los archivos el disco duro *****Comandos sumamente peligrosos de usar****** git reset con tres variantes --soft borra el historial y los registros de Git pero se guardan los cambios en staging, esto permite aplicar las ultimas actualizaciones a un nuevo commit --hard: El mas peligroso borra todo!!!, toda la informacion de los commits y del area del historial.

Comando de reporte

git status : muestra el estado de los archivos. git log : Muestra los cambios. git log --stat: nos muestra el historial de cambios desde el mas nuevo hasta mas anciano jajaja git show archivo: txt muestra los cambios hechos al archivo git diff muestas las diferecias entre la fabrica de trabajo y la sala de espera
